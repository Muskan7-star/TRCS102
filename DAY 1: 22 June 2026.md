# Day 1: MERN Stack Training

## Topic: Introduction to HTML
Today I started my MERN Stack training and learned the basics of HTML (HyperText Markup Language).

### Key Concepts:
* **HTML Tags**: Building blocks used to structure web pages.
* **Elements**: Usually consist of an opening tag, content, and a closing tag.
* **Attributes**: Provide additional information about elements (like `href` or `src`).

---

## HTML Basic Tags & Code Structure
Below is an example of the basic boilerplate structure and common elements:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Day 1: Basic HTML</title>
</head>
<body>

    <h1>This is a Heading 1 (Main Title)</h1>
    <h2>This is a Heading 2 (Subtitle)</h2>

    <p>This is a paragraph. HTML stands for <b>HyperText Markup Language</b>.</p>
    
    <h3>Things to learn in MERN Stack:</h3>
    <ul>
        <li>MongoDB (Database)</li>
        <li>Express.js (Backend Framework)</li>
        <li>React.js (Frontend Library)</li>
        <li>Node.js (Runtime Environment)</li>
    </ul>

</body>
</html>```
